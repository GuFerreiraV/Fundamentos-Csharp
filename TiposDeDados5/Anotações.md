## Struct DateTime
- Representa um momento no tempo expresso como uma data e hora.
- Uma variável do tipo DateTime é um tipo de valor e possui um valor padrão. 
- O valor padrão de um DateTime é 01/01/0001 00:00:00 

- ``DateTime dataAtual = DateTime.Now``; -> Sua representação fica assim: 25/03/2024 14:30:00
- ``DateTime data = new DateTime(2024, 03, 25);`` 
	- Comando usado para criar data e hora específicas.
	- operador ``new`` é usado para criar uma data específica.
	- Sua representação fica assim: 25/03/2024
